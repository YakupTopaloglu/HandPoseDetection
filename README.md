# HandPoseDetection
# Mekatronik Mühendisliği Tasarım Dersi Projesi
Bu proje, Mekatronik Mühendisliği Tasarım Dersi için geliştirilmiştir. Ana amaç, avuç içinde belirli noktaları tespit ederek bu noktaları belirlenen kare içine almak ve ardından arayüzde atanan komut tuşları ile fotoğraflarını çekip yerel olarak kaydetmektir.

# Proje Özellikleri
Kullanılan Teknolojiler:
MediaPipe
OS
OpenCV
Tkinter
Platform:

# Linux işletim sistemi bulunduran Raspberry Pi 4 üzerinde çalışacak şekilde optimize edilmiştir.

# Mechatronics Engineering Design Project
This project was developed for the Mechatronics Engineering Design Course. The primary objective is to design a 3D box in which specific points on the palm are captured within a designated square. These points are then photographed and saved locally through an interface using assigned command buttons.

# Project Features
Technologies Used:

MediaPipe
OS
OpenCV
Tkinter
Platform:

# Optimized to run on a Raspberry Pi 4 with a Linux operating system.
